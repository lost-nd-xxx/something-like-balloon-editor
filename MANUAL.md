# Something Like Balloon Editor 操作マニュアル

---

## 使い方

### 1. プロジェクトを用意する

このアプリはバルーン素材を「プロジェクト」として管理します。
プロジェクトとは `projects/（プロジェクト名）/` フォルダのことで、descript.txt・install.txt・画像ファイルなどが含まれます。

プロジェクトを用意する方法は3つあります。

#### 新規プロジェクトを作成する

メニューバーの「ファイル > 新規プロジェクト...」からプロジェクト名を入力して作成します。
空の descript.txt / install.txt が生成されます。

#### 既存フォルダからプロジェクトを作成する

メニューバーの「ファイル > フォルダからプロジェクトを作成...」でフォルダを選択します。
選択フォルダ内の PNG / PNA / PNR / TXT ファイルが新しいプロジェクトフォルダにコピーされます。

- Shift_JIS など UTF-8 以外のテキストファイルは自動的に UTF-8 に変換され、`descript.txt` / `install.txt` の `charset,` 宣言も UTF-8 に統一されます（宣言が無い場合は補完されます）。
- `descript.txt` に選択肢（cursor）・アンカー（anchor）の装飾設定がグループ単位で含まれていない場合、アプリの初期値が補完されます。補完が行われた場合は作成後に通知が表示されます。

取り込み元のフォルダ自体は変更されません（コピー先のプロジェクトフォルダのみ変換・補完されます）。

#### 既存プロジェクトを開く

メニューバーの「ファイル > プロジェクトを開く...」でプロジェクト一覧から選択して開きます。

プロジェクトを開いた際、`descript.txt` / `install.txt` の `charset,` 宣言が UTF-8 以外だった場合は自動的に UTF-8 へ修正され、通知が表示されます（ファイルの中身がそもそも UTF-8 として読み込めない場合は修正されず、警告のみ表示されます）。装飾設定の補完も同様にこのタイミングで行われます。

---

### 2. バルーン画像の配色を設定する

上部ツールバーのカラーピッカーでレイヤーカラーを指定します。

| レイヤー | 対象 |
|----------|------|
| 塗り | バルーン本体の背景色 |
| 縁取り | バルーンの縁・枠線 |
| 題字 | インプットボックスのテキスト（balloonc系のみ） |
| 部品 | 矢印・クリック待ち等のパーツ類 |

- カラーボタンをクリックするとカラーダイアログが開きます。
- テキストボックスに `#RRGGBB` 形式で直接入力してEnterキーを押すこともできます。
- **「色を一括適用」**: 有効にすると全バルーンに同じ配色を適用します。無効のときは選択中バルーンのみに適用されます。
- **「バルーン画像色を変更しない」**（編集メニュー）: 有効にするとバルーン画像への色オーバーレイをスキップし、元の画像色のまま出力します。有効時はツールバーのカラー関連コントロールがすべて無効化されます。

---

### 3. プレビューで確認する

左カラムのリストからバルーンを選択すると中央のプレビューに表示されます。

- **balloonc系（c0〜c4）**: 入力ボックス（communicatebox）のみ表示
- **balloons・balloonk・balloonp\*def系**: サンプルテキスト・矢印・マーカー類・カウンタ等を表示

プレビュー上部には現在の配色（バルーン色 vs 文字色）に基づいたコントラスト比が表示されます。WCAG2とAPCAの2種類の指標を同時に確認できます。

- **WCAG2**: 従来の比率ベース指標。**AA（4.5:1以上）** が一般的な合格基準、**AAA（7.0:1以上）** がより厳しい基準。
- **APCA Lc値 / Bronze**: より知覚特性に近い新世代指標。**Bronze Simple Mode** では **|Lc| ≥ 60** を合格（Bronze✓）とします。

**注意**: プレビューはSSP等のベースウェアの描画と完全に一致するものではありません。フォントのレンダリングや細部の表示は実際の動作環境で異なる場合があります。

---

### 4. 設定を編集する

右カラムのバルーン設定タブで各種設定を編集できます。

#### 基本情報

バルーン名・作者情報・更新URLなど。変更は `descript.txt` / `install.txt` に反映されます。

#### バルーン詳細設定

フォント・テキスト色・座標・スタイルなどをアコーディオン形式で編集できます。
変更はリアルタイムでプレビューに反映されます。

##### 選択肢・アンカーの「ラスタオペレーション」

選択肢（cursor）・アンカー（anchor）の各グループには **「ラスタオペレーション」** チェックボックスがあります。これは descript.txt の `blendmethod` 設定を切り替えるもので、強調表示の描画方式が変わります。

- **ON（`blendmethod,copypen`）**: 矩形の枠色（pen）・塗り色（brush）を個別に指定できます。文字色・枠色・塗り色の3つを編集します。
- **OFF（`blendmethod,none`）**: ラスタオペレーションを使わず、文字色と影色で表現します。文字色・影色・影スタイルを編集します。

`blendmethod` に `copypen` / `none` 以外の値（`mergepennot` など）が設定されている場合、このアプリでは描画を再現できないため、チェックボックスはグレーアウトされます（マウスを乗せると理由が表示されます）。

**注意**: `blendmethod,none`（ラスタオペレーションOFF）のとき、強調形状に矩形（square）を指定した場合のプレビューは、視認性確保のため文字色を反転した仮の色で塗りつぶして表示します。SSP での実際の表示とは異なります。

#### 個別設定

設定パネル上部に「個別設定」コントロールがあります。

- **「追加」ボタン**: 選択バルーン専用の設定ファイル（例: `balloonk0s.txt`）を作成します。追加後は設定タブの変更がそのバルーン専用に書き込まれ、他のバルーンには影響しません。
- **「削除」ボタン**: 個別設定を破棄し、共通設定（`descript.txt`）に戻します。
- 個別設定ファイルは `descript.txt` からの差分のみを保持します。

#### 位置編集

テキスト域・各マーカー・入力ボックスなどのUI要素の座標を、プレビュー上でドラッグして編集できます。

- 編集したい要素のボタンをクリックするとドラッグ編集モードになり、プレビュー上にハンドルが表示されます。
- ハンドルをドラッグすると座標が更新されます。
- **テキスト域オーバーレイ表示**（表示メニュー）: 位置編集モード以外でもオーバーレイを常時表示できます。

**注意**: SSTPメッセージ・カウンタのラベル表示には、フォントは反映されません。

---

### 5. 画像をプロジェクトに追加する

メニューバーの「ファイル > 画像をプロジェクトに追加...」で PNG 画像をインポートできます。プロジェクトを開いている状態でのみ実行できます。

- 画像をドラッグ＆ドロップしてインポートすることもできます。
- インポートダイアログでは、ファイル名から推測したカテゴリ（バルーン・矢印・マーカー等）やスコープ番号が自動で設定されます。必要に応じて変更してください。
- ダイアログ右側に取り込む画像のプレビューが表示されます（透過部分はチェッカー柄で表示）。
- 保存予定のファイル名が表示され、同名ファイルが既に存在する場合は上書き確認が出ます。
- `profile/slbe/files.txt` に既に定義済みのバルーン名をインポートすると、該当の定義行がコメントアウトされます（物理画像が優先されます）。

**注意**: 画像のインポートは「元に戻す（Ctrl+Z）」で取り消せません。

---

### 6. 保存する

メニューバーの「ファイル > 上書き保存（Ctrl+S）」で保存します。
以下のファイルが書き出されます。

- `descript.txt` / `install.txt`
- 個別設定ファイル（`balloonk0s.txt` など、差分がある場合のみ）
- `profile/slbe/profile.json`（配色設定）

未保存の変更がある状態でプロジェクトを切り替えたり終了しようとすると、確認ダイアログが表示されます。

**注意**: **「元に戻す（Ctrl+Z）」で戻せる操作はテキスト・色設定の変更のみです。画像のインポートは取り消せません。**

---

### 7. バルーンとして出力する

メニューバーの「ファイル > バルーンとして出力（Ctrl+E）」をクリックします。
`output/（インストール先フォルダ名）/` に以下のファイルが出力されます。

- バルーン画像（`balloonk0〜`、`balloons0〜`、`balloonp*def0〜`、`balloonc0〜4` など）
- パーツ画像（`arrow0/1`、`clickwait`、`online0〜`、`sstp`、`marker` など）
- `descript.txt` / `install.txt` / `readme.txt`
- 個別設定ファイル（差分が存在するバルーンのみ）

**注意**: 出力される `descript.txt` では `use_self_alpha`・`use_input_alpha`・`overlay_outside_balloon` が強制的に `1` で書き出されます。このアプリはPNGのアルファチャンネルを使って画像合成を行うため、これらを `0` にした場合の動作は保証されません。

## メニュー

### ファイルメニュー

| 項目 | 説明 |
|------|------|
| 新規プロジェクト... | 空のプロジェクトを新規作成します |
| プロジェクトを開く... | 既存プロジェクトを一覧から選択して開きます |
| フォルダからプロジェクトを作成... | 既存の素材フォルダを取り込んでプロジェクトを作成します |
| 上書き保存 (Ctrl+S) | 現在の編集内容をプロジェクトフォルダに保存します |
| 別名で保存... | 現在のプロジェクトを別名でコピー保存します |
| 画像をプロジェクトに追加... | PNG画像をプロジェクトにインポートします |
| プロジェクトをエクスプローラで開く | プロジェクトフォルダをエクスプローラで開きます |
| バルーンとして出力 (Ctrl+E) | バルーン画像・設定ファイルを output/ に出力します |

### 編集メニュー

| 項目 | 説明 |
|------|------|
| 元に戻す (Ctrl+Z) | 直前の操作を取り消します（最大20件）。画像インポートは取り消せません |
| やり直し (Ctrl+Y) | 取り消した操作をやり直します |
| バルーン設定を初期値に戻す | 配色・descript設定・個別設定をデフォルト値にリセットします |
| レイアウト定義を編集... | `profile/slbe/files.txt`（レイヤー合成定義）を GUI で編集します |
| バルーン画像色を変更しない | 有効にするとバルーン画像への色オーバーレイを無効化します |
| 奇数・偶数番バルーンを自動補完 | 有効にすると対応する偶数/奇数バルーンを反転して自動生成します |

### 表示メニュー

| 項目 | 説明 |
|------|------|
| プレビュー更新 (F5) | プレビューを強制再描画します |
| ウィンドウレイアウトをリセット | ペインの幅をデフォルトに戻します |
| テキスト域オーバーレイ表示 | テキスト域・折り返し位置のオーバーレイを常時表示します |
| テーマ | ライト／ダークテーマを切り替えます |
| プレビューテキスト | 通常サンプル／折り返し幅確認テキストを切り替えます |
| プレビュー背景 | プレビューキャンバスの背景をチェッカー（透明）または単色から選択します |

## profile/slbe/files.txt について

`profile/slbe/files.txt` はバルーン画像の合成レイアウトを定義するファイルです。
このファイルが存在する場合は**レイヤー合成モード**、存在しない場合は**画像編集なしモード**として動作します。

```
// コメント行（先頭が // の行は無視される）
balloonc0,c_edge.png:edge,c_base.png:base,c0_text.png:text
balloonk0,k0_edge.png:edge,k0_base.png:base
```

- 書式: `バルーン名,ファイル名:色種別,...`（奥側レイヤーから順に記述）
- 色種別: `edge`（縁取り色）/ `base`（塗り色）/ `text`（テキスト枠色）/ `none`（塗り替えなし）
- `balloonk1/k3`・`balloons1/s3` などは記述を省略すると、対応する偶数バルーンを左右反転して自動生成します（「奇数・偶数番バルーンを自動補完」が有効な場合）。
- ファイルが存在しない・必須バルーンが未定義など、記述に問題がある場合は起動時にエラーが表示されます。

### GUI で編集する

メニューバーの「編集 > レイアウト定義を編集...」で、`profile/slbe/files.txt` を専用ウィンドウで編集できます。

- バルーンごとに、合成するレイヤー（ファイル名と色種別）を追加・削除・並べ替えできます。
- 各バルーンの種別（通常系 `balloons*` / 相方系 `balloonk*` など、入力ボックス `balloonc*`）を切り替えられます。
- コメント行の追加・一括削除、バルーン順への並べ替えができます。
- 「適用」すると変更がプレビューに反映され、保存時に `profile/slbe/files.txt` へ書き出されます。空のコメント行は書き込み時に空行として出力されます。

### 旧形式からの自動移行

素材フォルダ直下の `files.txt`（旧形式）が存在し、このアプリのレイアウト定義として解釈できる場合は、`profile/slbe/files.txt` へ自動的に移行します。

---

## 画像編集なしモード

`profile/slbe/files.txt` のないプロジェクトは画像編集なしモードで動作します。
`balloon*.png` ファイルをそのままプレビュー・出力します。バルーン画像への色オーバーレイは行われません。

---

## プロジェクトのフォルダ構成

```
projects/
  （プロジェクト名）/
    descript.txt          … バルーン設定（テキスト正本）
    install.txt           … インストール設定（テキスト正本）
    readme.txt            … 説明文（任意）
    balloon*.png          … バルーン・パーツ画像
    balloon*s.txt         … 個別バルーン設定（差分のみ、任意）
    profile/
      slbe/
        files.txt         … レイヤー合成定義（任意、なければ画像編集なしモード）
        profile.json      … 配色設定（このアプリ固有）
```

`profile/slbe/` 以下はこのアプリ固有のファイルです。SSPには配布不要です（出力時には含まれません）。
